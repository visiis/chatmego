# IM系统UI设计文档

## 1. 设计风格

### 1.1 整体风格
- 仿照微信的设计风格，简洁、现代、易用
- 主色调：粉红系，体现社交，爱情和活力
- 辅助色：白色、浅灰色，营造清爽的视觉效果
- 字体：强制使用思源黑体，保证在不同设备上的一致性
- 图标：使用PNG图标，后台可以上传更改，保持视觉统一
- 国际化，字体使用繁体，主要地区为台湾，多国语言支持

### 1.2 布局原则
- 顶部导航栏固定，底部标签栏固定
- 内容区域可滚动
- 卡片式设计，清晰分隔不同功能模块
- 响应式布局，适配不同屏幕尺寸

## 2. 主要页面设计

### 2.1 登录/注册页
- **页面背景**：
  - 全屏背景图，后台可配置上传
  - 支持多张背景图随机显示
  - 背景图可设置是否显示遮罩层（深色/浅色）
  - 背景图可设置模糊程度
  - 默认提供几张精美背景图
- **登录页**：
  - 顶部Logo区域（可配置显示/隐藏）
  - 手机号输入框
  - 密码输入框
  - 验证码登录选项
  - 忘记密码链接
  - 登录按钮
  - 注册入口
  - 第三方登录（微信、Apple ID等，可选）
- **注册页**：
  - 与登录页相同的背景图配置
  - 手机号输入框
  - 验证码输入框
  - 密码设置输入框
  - 邀请码输入框（可选）
  - 注册按钮
  - 用户协议和隐私政策链接

### 2.2 主页面
- **底部标签栏**：
  - 发现（卡片方式，左边不喜欢，右边喜欢，可以左右滑动选择）
  - 通讯录
  - 聊天（通讯录和聊天界面）
  - 我
- **消息页面**：
  - 消息列表，显示最近聊天
  - 每个消息项包含：头像、昵称、最新消息、时间、未读消息数
  - 顶部搜索栏
  - 右上角+按钮（发起聊天、添加好友等）

### 2.3 聊天页面
- **顶部**：对方昵称、返回按钮、更多选项
- **消息区域**：
  - 消息气泡（自己的在右，对方的在左）
  - 支持文本、图片、语音、视频、表情、礼品等消息类型
  - 消息时间戳
  - 已读状态显示
- **底部输入栏**：
  - 文本输入框
  - 表情按钮
  - 礼品按钮（需要积分购买，发送一次扣除一次积分）
    - 点击礼品按钮弹出礼品选择面板
    - 显示当前积分余额
    - 礼品按分类展示（热门、新品、专属等）
    - 每个礼品显示所需积分和预览图
    - 积分不足时提示充值或购买会员
    - 发送后播放礼物动画效果
  - 图片/视频选择按钮
  - 语音输入按钮
  - 更多功能按钮（文件、位置等）

### 2.4 通讯录页面
- **顶部**：搜索栏
- **功能入口**：
  - 新的朋友
  - 群聊
  - 标签
  - 公众号
- **好友列表**：按字母顺序排列，右侧有字母索引
- **好友项**：头像、昵称、备注

### 2.5 发现页面
- **卡片展示内容**:
  - 用户第一张照片（大图全屏展示）
  - 底部信息栏：昵称、年龄、地区、距离
  - 个性签名（限制字数，显示在底部）
  - 兴趣爱好标签（底部显示）
  - 照片指示器：显示当前是第几张/总张数
- **照片浏览交互**：
  - **上滑**：查看下一张照片
    - 循环浏览用户的所有照片
    - 包括普通照片和付费照片
    - 最后一张照片后回到第一张
  - **下滑**：查看上一张照片
    - 反向循环浏览
  - **付费照片展示**：
    - 显示模糊预览图
    - 底部显示"🔒 付费查看"按钮
    - 点击按钮弹出购买确认框
    - 确认后扣除积分，显示清晰原图
    - 购买后该照片永久解锁
  - **已购买照片**：直接显示清晰原图，显示"已解锁"标识
- **用户操作交互**：
  - **左滑**：不喜欢 ❌
    - 卡片向左飞出
    - 减少该类型用户的推荐权重
    - 记录不喜欢原因（可选）
  - **右滑**：喜欢 ❤️
    - 卡片向右飞出
    - 增加该类型用户的推荐权重
    - 如果对方也喜欢，触发匹配
  - **点击**：查看详细资料
    - 跳转用户详情页
    - 显示完整资料和所有照片
- **推荐算法反馈**：
  - 喜欢的用户特征：年龄、地区、兴趣、外貌等
  - 不喜欢的用户特征
  - 实时调整推荐列表
- **匹配成功**：
  - 弹出匹配成功提示框
  - 显示双方头像
  - 提供"立即聊天"和"继续浏览"按钮
  - 播放匹配成功音效和动画
- **筛选功能**：
  - 按年龄范围筛选
  - 按地区筛选
  - 按性别筛选
  - 按兴趣爱好筛选
  - 筛选条件保存和重置
- **每日限制**：
  - 显示今日剩余喜欢次数
  - 会员可增加次数
  - 积分可购买额外次数

### 2.6 用户详细资料页（个人空间）
- **顶部照片墙区域**：
  - 全屏照片墙，类似发现页面的卡片展示
  - 左右滑动浏览所有照片
  - 照片指示器显示当前位置
  - **普通照片**：正常显示
  - **付费照片**：
    - 模糊显示
    - 显示"🔒 付费查看"按钮
    - 点击购买后显示清晰原图
  - **已购买照片**：显示清晰原图 + "已解锁"标识
  - 底部渐变遮罩，显示基本信息
- **用户信息区域**：
  - 头像（小圆图）+ 昵称 + 会员等级标识
  - 在线状态指示器
  - 关注/喜欢按钮
  - 发送消息按钮
- **详细资料卡片**：
  - 基础信息：年龄、性别、地区、距离
  - 个人标签：兴趣爱好、性格特点
  - 详细信息：身高、体重、星座、血型等
  - 爱情宣言（大字体突出显示）
  - 理想对象描述
- **互动数据**：
  - 被喜欢次数
  - 匹配成功率
  - 照片被购买次数（如果是付费照片）
- **操作按钮**：
  - 喜欢/取消喜欢
  - 发送消息
  - 举报/拉黑
  - 分享个人主页
- **底部推荐**：
  - "你可能感兴趣的人"
  - 横向滑动推荐其他用户

### 2.7 个人中心页面
- **顶部**：头像、昵称、微信号、二维码
- **功能模块**：
  - 我的相册
  - 我的收藏
  - 我的积分（积分可以送礼物）
    - 积分余额展示（大字体显示）
    - 积分获取方式：
      - 每日签到（连续签到有额外奖励）
      - 完成任务（完善资料、邀请好友等）
      - 充值购买（多种充值套餐）
      - 会员赠送（会员每日领取积分）
    - 积分明细页面：
      - 收入明细（签到、任务、充值等）
      - 支出明细（送礼物、购买特权等）
      - 时间筛选和搜索
    - 积分兑换：
      - 兑换礼品
      - 兑换会员时长
      - 兑换喜欢次数
  - 我的礼物（收到的礼物可以按照50%比例换积分）
    - 收到的礼物列表
    - 每个礼物显示赠送者、时间、价值
    - 一键兑换积分功能
    - 礼物展示柜（可设置公开/私密）
  - 我的相册
    - 普通照片管理
    - 付费照片管理
      - 设置照片价格
      - 查看购买记录
      - 查看收入统计
      - 已售照片显示购买者列表
  - 照片购买记录
    - 我购买的照片列表
    - 可查看原图
    - 购买时间记录
  - 我的小程序
  - 设置
- **会员中心入口**：显示会员等级和特权

## 3. 会员系统UI

### 3.1 会员中心
- **会员等级展示**：当前等级、经验值、升级进度
- **会员特权列表**：
  - **普通会员（免费）**：
    - 每日10次喜欢机会
    - 基础聊天功能
    - 查看用户基础资料
  - **银卡会员**（月费30元）：
    - 每日50次喜欢机会
    - 优先匹配推荐
    - 查看谁喜欢我
    - 每日领取100积分
    - 专属银卡标识
  - **金卡会员**（月费68元）：
    - 无限喜欢次数
    - 超级喜欢每日10次
    - 优先展示在推荐列表
    - 每日领取300积分
    - 专属金卡标识和头像框
    - 消息已读功能
  - **钻石会员**（月费128元）：
    - 所有金卡特权
    - 隐身浏览模式
    - 消息撤回功能
    - 每日领取800积分
    - 专属钻石标识和动态头像框
    - 专属客服通道
    - 线下活动优先参与权
- **会员续费/升级**：不同会员套餐选择
  - 月卡、季卡、年卡选项
  - 连续包月优惠
  - 升级补差价功能
- **会员活动**：限时优惠、活动通知

### 3.2 礼品系统
- **礼品商城**：各类礼品展示，包含价格和描述
- **礼品赠送**：选择好友、选择礼品、添加留言
- **礼品记录**：赠送和接收的礼品历史

## 4. 后台管理系统UI

### 4.1 登录页
- 管理员账号密码输入
- 验证码
- 登录按钮

### 4.2  dashboard
- 系统概览：用户数、消息数、礼品交易数等统计
- 快捷操作入口
- 系统状态监控

### 4.3 用户管理
- 用户列表：支持搜索、筛选、分页
- 用户详情：查看和编辑用户信息
- 用户操作：禁用、启用、重置密码等

### 4.4 会员管理
- 会员等级设置：添加、编辑、删除会员等级
- 会员特权配置：为不同等级设置不同特权
- 会员计费管理：可以添加或者减少用户积分
  - 批量操作用户积分
  - 积分变动记录查询
  - 积分补偿机制
- 会员统计：会员分布、续费情况等

### 4.5 礼品管理
- 礼品列表：添加、编辑、删除礼品
- 礼品分类：管理礼品分类
- 礼品动画管理：
  - 上传礼物动画文件（GIF、Lottie、视频等）
  - 预览礼物动画效果
  - 设置动画播放时长和音效
  - 关联动画与礼品
- 礼品交易记录：查看礼品赠送和接收记录

### 4.6 系统设置
- 基本设置：系统名称、Logo、联系方式等
- 安全设置：密码策略、登录限制等
- 通知设置：系统通知、邮件模板等
- 多语言管理：
  - 支持语言列表管理（繁体中文、简体中文、英文、日文等）
  - 界面文本翻译管理
  - 动态切换语言
  - 翻译进度统计
- 积分规则设置：
  - 签到积分配置
  - 任务积分配置
  - 充值比例配置
- 匹配算法设置：
  - 推荐权重配置
  - 筛选条件配置
- 照片售卖设置：
  - 开启/关闭照片售卖功能
  - 平台抽成比例设置
  - 照片价格范围设置
  - 照片审核管理

## 5. 响应式设计

### 5.1 小程序适配
- 适配微信小程序的屏幕尺寸
- 遵循小程序的设计规范
- 优化触摸交互

### 5.2 APP适配
- 适配iOS和Android不同屏幕尺寸
- 支持横屏和竖屏模式
- 优化不同设备的性能

## 6. 交互设计

### 6.1 动画效果
- 页面切换动画：平滑过渡
- 消息发送动画：气泡弹出效果
- 礼物发送动画：播放动画效果
- 加载动画：骨架屏、加载指示器
- 操作反馈：按钮点击效果、消息送达提示

### 6.2 手势操作
- 下拉刷新：刷新消息列表
- 上拉加载：加载更多消息
- 左滑操作：消息置顶、删除等
- 长按操作：消息多选、复制、转发等

### 6.3 导航设计
- 底部标签栏导航
- 顶部导航栏返回和操作
- 侧滑菜单（可选）
- 面包屑导航（后台管理）

### 6.4 卡片浏览交互
- **照片浏览手势**：
  - **上滑**：查看下一张照片
    - 平滑过渡动画
    - 显示照片切换指示器
    - 循环浏览所有照片
  - **下滑**：查看上一张照片
    - 反向浏览照片
    - 支持快速滑动
- **付费照片购买**：
  - 付费照片默认模糊显示
  - 底部显示"🔒 付费查看"按钮
  - 点击弹出购买确认弹窗
    - 显示照片预览（小图）
    - 显示价格
    - 显示当前积分余额
    - "确认购买"和"取消"按钮
  - 购买成功后立即显示清晰照片
  - 显示"已解锁"标识
- **用户操作手势**：
  - **左滑**：不喜欢
    - 卡片向左飞出动画
    - 显示"不喜欢"图标
    - 减少该类型推荐
  - **右滑**：喜欢
    - 卡片向右飞出动画
    - 显示"喜欢"图标
    - 增加该类型推荐
  - **点击**：查看详情
    - 进入用户详细资料页

### 6.5 照片售卖交互
- **上传照片流程**：
  - 选择照片后弹出设置选项
  - 选择"普通照片"或"付费照片"
  - 付费照片需设置积分价格（有最小/最大限制）
  - 系统自动生成模糊预览图
  - 上传成功后显示在相册中
- **购买照片流程**：
  - 在卡片浏览或用户资料页看到付费照片
  - 点击"付费查看"按钮
  - 弹出购买确认框
  - 显示照片价格和当前积分余额
  - 确认购买后扣除积分
  - 显示原图并保存到购买记录
  - 照片所有者收到积分到账通知
- **照片收入查看**：
  - 个人中心显示照片收入统计
  - 可查看每张照片的购买次数和收入
  - 收入明细列表

### 6.5 空状态和异常状态
- **空状态页面**：
  - 无网络连接：显示网络错误图标和重试按钮
  - 无消息：显示空消息图标和引导提示
  - 无好友：显示添加好友引导
  - 无匹配：显示完善资料引导
  - 无礼品：显示礼品商城入口
  - 无积分记录：显示获取积分引导
  - 无购买记录：引导去浏览用户资料
- **异常状态页面**：
  - 404页面：页面不存在提示
  - 500页面：服务器错误提示
  - 加载失败：重试机制和错误提示
  - 权限不足：提示升级会员
  - 账号被封禁：显示封禁原因和解封时间
- **加载状态**：
  - 骨架屏：页面内容加载时的占位效果
  - 加载指示器：圆形进度条
  - 下拉刷新：下拉时的刷新动画
  - 上拉加载：上拉时的加载动画

## 7. 视觉层次

### 7.1 色彩层次
- 主色调：用于品牌标识、主要按钮
- 辅助色：用于次要按钮、提示信息
- 中性色：用于背景、文本
- 功能色：用于成功、警告、错误等状态

### 7.2 字体层次
- 标题：较大字号，粗体
- 正文：中等字号，常规
- 辅助文本：较小字号，浅色

### 7.3 空间层次
- 卡片间距：保持一致的间距
- 内容区域：适当的内边距
- 元素对齐：保持视觉对齐

## 8. 设计规范

### 8.1 图标规范
- 图标风格：PNG图标，后台上传
- 图标大小：统一尺寸，根据不同位置调整
- 图标颜色：根据功能和状态使用不同颜色

### 8.2 按钮规范
- 主要按钮：填充色，圆角设计
- 次要按钮：边框色，圆角设计
- 文字按钮：无背景，仅文字
- 按钮尺寸：根据功能和位置调整

### 8.3 表单规范
- 输入框：圆角设计，有聚焦状态
- 标签：清晰的标签文字
- 错误提示：红色文字，位于输入框下方
- 提交按钮：突出显示

### 8.4 推送通知UI
- **应用内通知**：
  - 新消息通知：顶部横幅显示发送者和消息预览
  - 匹配成功通知：全屏弹窗显示匹配信息和操作按钮
  - 系统通知：列表形式展示，支持标记已读
  - 礼物接收通知：动画效果展示礼物和赠送者
- **推送通知样式**（手机系统通知）：
  - 新消息：显示发送者头像、昵称和消息内容摘要
  - 匹配成功：显示"你与XXX互相喜欢，开始聊天吧"
  - 礼物提醒：显示"XXX送给你一份礼物"
  - 积分到账：显示积分变动详情
  - 会员到期提醒：提前3天、1天、当天提醒

### 8.5 搜索功能
- **全局搜索**：
  - 搜索栏位置：顶部导航栏
  - 搜索范围：好友、聊天记录、朋友圈
  - 搜索结果分类展示
  - 搜索历史记录
  - 热门搜索推荐
- **好友搜索**：
  - 按昵称、备注搜索
  - 按手机号搜索（需权限）
  - 搜索结果展示头像、昵称、地区
- **消息搜索**：
  - 按关键词搜索聊天记录
  - 搜索结果展示上下文
  - 支持按时间筛选
- **发现页面搜索**：
  - 按条件筛选用户
  - 高级筛选（年龄、地区、兴趣等）
